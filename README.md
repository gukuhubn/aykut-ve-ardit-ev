# Aykut ve Ardit Ev — Proje Portalı (yayın paketi)

Sundurma → vestiyer dönüşümü, Orange (84100), Fransa. Canlı portal:
https://gukuhubn.github.io/aykut-ve-ardit-ev/

Bu repo yalnız yayına giden statik paketi içerir (portal + PDF/DXF
dokümanlar + fotoreal görseller). Kaynak repo özeldir. İhale/ticari
dosyalar burada YOKTUR ve asla eklenmez (otomatik gizlilik kapısı,
içerik taramalı, her yayında koşar).

Ölçü ve imalat otoritesi çizimlerdedir; 3B model ve fotoreal görseller
illüstratiftir. İmalat öncesi yerinde ölçü zorunludur.
